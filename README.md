# Primordial Engine
Primordial Engine is a bare-bones 2D game engine built with GLFW and OpenGL<br />

## Core Features
- Window Management<br />
- Input System<br />
- Audio System<br />
- 2D Rendering<br />
- GUI<br />
- Entity Component System<br />
- Physics<br />
- Framerate independant logic<br />
